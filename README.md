# Cobra
My own programming language
# What is Cobra?
Cobra is made using Python and is essencially a easier version of Python which can be used for begginers.
# Commands
# Say:
Say: is the print command for Cobra.
Say: will print a word of choice onto the console
# SayVar:
SayVar: will print a variable onto the console.
# SaveVar:
SaveVar: will save the input to a variable
After the colon all you have to do is type what you want the variable to be called
